code-to-prompt

A CLI tool to gather and format files for LLM prompts

Features

  • Recursively scan directories and gather files
  • Respects .gitignore and custom ignore patterns
  • Filter by extension, include hidden files, add line numbers
  • Output formats: Plain text, Markdown, Claude‑XML
  • Optional file‑tree overview

Installation

npm install -g code-to-prompt

Usage

code-to-prompt [options] [paths...]

Examples:

// Process current directory
code-to-prompt .

// Only .ts and .js in src
code-to-prompt -e .ts -e .js ./src

// Markdown output & ignore node_modules via .gitignore
code-to-prompt ./src --markdown -o prompt.md
